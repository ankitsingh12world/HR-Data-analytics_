# 📊 HR Attendance Analytics Dashboard (Power BI)

## 📌 Project Overview
This project analyzes employee attendance data to track workforce availability, leave patterns, and productivity trends.  
The dashboard is built using **Power BI** and helps HR teams monitor employee attendance behavior efficiently.

---

## 🎯 Problem Statement
Managing employee attendance manually makes it difficult to:
- Track presence and absenteeism accurately
- Analyze leave patterns
- Monitor work-from-home trends
- Identify employee availability

This dashboard provides a **centralized and interactive solution** to these problems.

---

## 📂 Dataset Details
- Source: Excel Attendance Sheet (2022–2023)
- Level: Employee-wise daily attendance
- Key Fields:
  - Employee Code
  - Employee Name
  - Daily Attendance (P, WO, SL, etc.)
  - Total Present Days
  - Work From Home (WFH)
  - Paid Leave (PL)
  - Sick Leave (SL)
  - Leave Without Pay (LWP)
  - Weekly Off (WO)
  - Holidays (HO)

---

## 📊 Key KPIs
- Total Employees
- Total Present Days
- Attendance Rate (%)
- Total Leaves Taken
- Work From Home Days
- Leave Without Pay Count

---

## 📈 Dashboard Features
- 📅 Daily Attendance Tracking
- 👥 Employee-wise Attendance Summary
- 🏠 Work From Home Analysis
- 🏖️ Leave Type Distribution (PL, SL, LWP, etc.)
- 📊 Attendance Trends Over Time
- 📌 Weekly Off & Holiday Analysis
- 🎛️ Interactive Filters (Employee, Date)

---

## 🔍 Key Insights
- Employees show consistent presence except during **weekly offs and holidays**
- Work-from-home usage varies across employees
- Some employees have higher **leave frequency**, which can impact productivity
- Leave without pay (LWP) indicates potential attendance issues
